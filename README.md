## Как использовать данный репозиторий

Делай следующие команды в данной последовательности:

```bash
git clone https://github.com/golzi99/next-starter my-new-project
cd my-new-project
rm -rf .git
pnpm install
```
Что делает эта последовательность:

* clone — копирует код.

* rm -rf .git — удаляет историю коммитов, чтобы ты мог начать с нуля.

* pnpm install — устанавливает зависимости.

## Варинат с GitHub Template

Если ты на GitHub нажмёшь кнопку "Use this template" (или включишь репозиторий как Template), то можешь создавать проекты через GitHub UI или по API:

Включить шаблон:

* Перейди на страницу репозитория.

* Нажми Settings → поставь галку Template repository.

После этого любой сможет нажать Use this template и создать проект.
